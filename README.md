# chess openings viz


hierarchical file structure

id: eco abbreviation
path: path to svg on path
name: canonical opening name
hm: amount halfmoves to reach position
children: array of objects

Example:

{
    "id": "A09",
    "path": "./img/A09.svg",
    "name": "Reti Opening",
    "hm": 4,
    "children": []
}
